# SmartVenue AI-Agent
### Next-Gen Stadium Experience Powered by Gemini AI

**SmartVenue AI** is a real-time venue management dashboard designed to optimize the fan experience and enhance safety during large-scale events. Built for the **Google PromptWar Virtual Challenge**, this agent leverages Google's Gemini 1.5 Flash to provide instant, context-aware recommendations for crowd control, emergency coordination, and queue management.

---

## 🚀 Key Features

* **Live Crowd Redirect:** Monitors venue congestion and suggests optimal routes to "Low" density zones to prevent bottlenecks.
* **Queue Predictor:** Uses AI to estimate wait times for gates and concessions, recommending the shortest path for fans.
* **Emergency Coordinator:** A dedicated module for Medical, Fire/Safety, and Lost & Found situations, providing instant AI-driven protocols for staff.
* **Multilingual Support:** Seamlessly switch between English, Hindi, and Marathi to assist diverse stadium crowds.
* **Secure API Configuration:** Built-in client-side key management using `localStorage` to ensure user privacy and security.

---

## 🛠️ Technical Stack
* **Environment:** Developed using **Google Antigravity** (Official PromptWar IDE)
* **Model:** Google Gemini 1.5 Flash
* **Frontend:** HTML5, CSS3 (Modern Grid & Flexbox), JavaScript (ES6+)
* **Architecture:** Serverless Static Deployment
* **DevOps:** Docker-ready (via `Dockerfile`) for containerized scaling.
* **Hosting:** Deployed on GitHub Pages with Google Cloud Run readiness.

## 📦 Local Installation & Deployment

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/ApurvaPatil2401/SmartVenue-AI-Agent.git](https://github.com/ApurvaPatil2401/SmartVenue-AI-Agent.git)

2. Open the project:
    Locate the folder and simply double-click index.html to open it in your preferred web browser.

3. Configure Gemini:
    To activate the AI features, paste your Gemini API Key into the configuration input at the top of the dashboard and click "Save Key".
(Note: Your key is stored locally in your browser and is never sent to any external server other than Google's Gemini API.)
